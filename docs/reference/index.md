# Elastic CLI Reference

Interface with the Elastic Stack and Elastic Cloud from the command line.

## Command Groups

| Group | Description |
|-------|-------------|
| [`version`](./version.md) | Print the elastic CLI version |
| [`stack es`](./stack-es.md) | Interact with the Elasticsearch API |
| [`stack kb`](./stack-kb.md) | Interact with the Kibana API |
| [`cloud accounts`](./cloud-accounts.md) | Cloud accounts commands |
| [`cloud authentication`](./cloud-authentication.md) | Cloud authentication commands |
| [`cloud organizations`](./cloud-organizations.md) | Cloud organizations commands |
| [`cloud user-role-assignments`](./cloud-user-role-assignments.md) | Cloud user-role-assignments commands |
| [`cloud hosted`](./cloud-hosted.md) | Manage Elastic Cloud Hosted deployments |
| [`cloud serverless`](./cloud-serverless.md) | Manage Elastic Serverless projects and resources |
| [`docs`](./docs.md) | Search, read, and ask questions about Elastic documentation |

See also: [Global Options](./global-options.md)
